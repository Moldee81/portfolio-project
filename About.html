Hi I am Anita Peña-Tomczak

Frontend Developer, based in Gran Canaria

I’m a newly minted frontend developer living on the beautiful, sunny island of Gran Canaria. My journey into programming began in November 2024, and since then I’ve discovered how rewarding it can be to create interfaces that combine aesthetics with functionality. I started by learning HTML and CSS — this very website you’re navigating was designed and built by me from scratch using those technologies. Over time, JavaScript opened up a world of interactivity and logic, allowing me to build clean, dynamic and engaging user interfaces. Next, I explored Python to gain a better understanding of backend principles, and soon after I moved on to React — a key technology in 2025 that enables the creation of efficient, component-based applications supported by a vast community. I love learning new things and constantly improving my skills. Along the way, I’ve been building new projects, experimenting with Bootstrap, and working with VS Code and GitHub. Beyond coding, I’m passionate about dance — especially salsa and bachata — which is why you’ll also find a page on this site dedicated to Latin dances.
